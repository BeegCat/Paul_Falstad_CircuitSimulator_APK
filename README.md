# Paul_Falstad_CircuitSimulator_APK

This is a FOSS android port of @pfalstad famous circuit simulator. The app is supposed to be android compatible. Feel free to ping me for an iOS version. 

# Issues
There may be some issues with layout of certain entities, and hence it is recommended to use it in **landscape** mode. On older devices, it may take some time to load.

# How to Download?
https://bit.ly/4bB29w7
click the above link and then click the "download apk" button.
You may need to enable "installation from third party sources" in your settings.
